Félix Bastian — Landing Page

Descrição
---------
Página simples de portfólio/landing page para serviços de desenvolvimento, suporte e automação.

Como abrir no VS Code
----------------------
1. Extraia o arquivo ZIP (se aplicável).
2. Abra a pasta `felix-landing-page` no VS Code.
3. Para visualização local rápida, instale a extensão Live Server e abra `index.html` com ela.

Como rodar localmente
---------------------
- Abra `index.html` diretamente no navegador.
- Ou rode um servidor simples (recomendado):

	```bash
	# Python 3
	python -m http.server 8000
	# então abra http://localhost:8000
	```

Arquivos principais
------------------
- `index.html`: estrutura da página
- `style.css`: estilos e responsividade
- `script.js`: comportamento do menu mobile
- `assets/`: imagens (banner, foto)

Editar contatos
---------------
Procure no `index.html` a seção `id="contato"` e substitua os links `#` pelos seus links reais.

Deploy (GitHub Pages)
---------------------
- Repositório remoto: https://github.com/FXBastian/felix-landing-page
- Para publicar no GitHub Pages, ative Pages nas configurações do repositório e escolha a branch `main`.
